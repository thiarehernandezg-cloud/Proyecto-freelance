Portafolio Personal
Este es un proyecto que facilita la comunicación con algún posible cliente o reclutador, cuenta un poco sobre mi y mis habilidades, finalmente tiene un formulario de contacto, mediante el cual pueden hacer llegar sus proyectos en mente para desarrollarlos.
Usé:
HTML5: para una estructura semántica y accesible
CSS3: para un diseño responsivo y estilo único 
JavaScript: Logica de interacción y validación del formulario
